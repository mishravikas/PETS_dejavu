History sniffing using Cache Fingerprinting

This page shows an application of browser-cache-abuse for history sniffing presented in our paper. The demo relies on the Tranco top 100 websites and tries to detect if and when those sites were visited by you.
The following table lists the websites along with their tranco list in one column and the Date when that particular website was visited on the Date column.
Tranco Rank Website Visited On Vulnerability Window

Instructions

  • Verify if the Dates shown in the table correspond with the actual time of visit.
  • Table will display current timestamp for the websites which have not been visited before.
  • Open the demo page in incognito mode and you should see the current timestamps
  • Closing the browser/tab and opening the page again should not change the timestamps
  • This demo relies on the browser cache so make sure you do not flush your cache before reloading the page.
  • If your dev tools is open, make sure Disable cache is not ticked.

This demo has been tested on Firefox, Chrome, Brave, Safari desktop and mobile browsers.
In Firefox, reloading the page by pressing Ctrl+R or pressing on the reload button next to the address bar will not work.
Try clicking on the adress bar and hitting the enter/return key or closing the tab/browser and reopening the demo page again.
